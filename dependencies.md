## Installation

```bash
bun install
```

## Adding Dependencies

```bash
# Runtime
bun add <package>

# Dev
bun add -d <package>
```

## Upgrading

```bash
bun update
```

Or specific:
```bash
bun update katex marked citty
```

## Removing

```bash
bun remove <package>
```

## Lock File

We use `bun.lock` (generated by `bun install`). This is intentional — Bun's lockfile is faster and more reliable than npm/yarn/pnpm alternatives for Bun projects.
